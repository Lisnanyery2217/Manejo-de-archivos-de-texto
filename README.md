# Manejo-de-archivos-de-texto


 Descripción:

Desarrollo de una aplicación en C# con Windows Forms que permite registrar datos de empleados mediante un formulario.
La aplicación valida campos vacíos, datos numéricos y selecciones en ComboBox, mostrando mensajes de advertencia y confirmación al usuario.
Los datos ingresados se visualizan en un DataGridView y pueden ser almacenados en un archivo de texto (.txt) utilizando manejo de archivos, permitiendo además abrir el archivo una vez guardado.
Este proyecto aplica validaciones, cuadros de diálogo y uso de archivos de texto para la gestión de información.
